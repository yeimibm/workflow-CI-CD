Despliegue automatico hacia VPS

Esta mini home page se valida, se construye como sitio estatico y se publica automaticamente en la VPS cuando hay un push a la rama principal.

Esperando informacion del navegador...

Flujo automatizado

  1. 1 Push a GitHub

    El cambio en la rama principal activa GitHub Actions.

  2. 2 Validacion

    El pipeline revisa que existan HTML, CSS y JavaScript.

  3. 3 Construccion

    Los archivos estaticos se copian a la carpeta dist.

  4. 4 Despliegue

    GitHub Actions copia el artefacto hacia /var/www/html.
